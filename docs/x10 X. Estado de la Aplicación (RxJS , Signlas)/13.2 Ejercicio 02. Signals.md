# Estado reactivo 

A diferencia del enfoque tradicional basado en observables y `ChangeDetection`, Signals ofrece una API mÃ¡s intuitiva y performante para manejar cambios de estado y actualizaciones de la interfaz de usuario.

## 1. Contador simple

1. Actualizar el componente `RegistroComponent``src\app\auth\registro\registro.component.ts`
Importar las librerias 
```javascript
import { Signal, signal } from '@angular/signaling';
```

Agregar el objeto signal
```javascript
export  class  RegistroComponent {
	count  =  signal(0);
	increment() {
		this.count.update(count  =>  count  +  1);
	}
}
```

2. Actualizar la plantilla del componente RegistroComponent
```javascript
<p>Contador: {{  count  }}</p>
<button  (click)="increment()">Incrementar</button>
```

3. Abrir el navegador 
http://localhost:4200/auth/registro
